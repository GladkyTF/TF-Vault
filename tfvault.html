⚡ JAK UZYSKAĆ KLUCZ GROQ (za darmo, 1000 req/dzień):

1. Wejdź na console.groq.com
2. Kliknij "Sign Up" → zarejestruj się e-mailem lub Google (bez karty!)
3. Po lewej → "API Keys" → "Create API Key"
4. Skopiuj klucz (gsk_...) i wklej powyżej

Model vision: Llama 3.2 Vision (90B) — bardzo dobra jakość rozpoznawania
Limit: 1000 req/dzień — resetuje się codziennie o północy UTC
★ JAK UZYSKAĆ KLUCZ OPENROUTER (zapasowy):

1. Wejdź na openrouter.ai
2. Kliknij "Sign in" → zarejestruj się e-mailem (bez karty!)
3. Menu górne → "Keys" → "Create Key"
4. Skopiuj klucz (sk-or-v1-...) i wklej powyżej
JAK UZYSKAĆ KLUCZ GEMINI (bezpłatnie):

1. Wejdź na aistudio.google.com
2. Zaloguj się kontem Google
3. Kliknij "Get API key" → "Create API key"
4. Skopiuj klucz (AIza...) i wklej powyżej

Limit: 15 req/min, 1500/dzień na klucz

TIP: Możesz dodać 3 klucze z 3 różnych kont Google — aplikacja automatycznie przełączy się na następny gdy jeden osiągnie limit (razem 4500 zapytań/dzień).

WYCENA: Dane z zakończonych aukcji eBay przez darmowy proxy — bez klucza API.
