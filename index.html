JAK UZYSKAĆ KLUCZE API (bezpłatnie):

Gemini (rozpoznawanie):
1. Wejdź na aistudio.google.com
2. Zaloguj się kontem Google
3. Kliknij „Get API key" → „Create API key"
Limit: 15 req/min, 1500/dzień – wystarczy

PriceCharting (wycena):
1. Zarejestruj się na pricecharting.com
2. Ustawienia konta → sekcja API
Podstawowy dostęp jest bezpłatny
JAK ZAINSTALOWAĆ NA TELEFONIE:

Opcja 1 – GitHub Pages (darmowy, bez rejestracji karty):
1. Utwórz konto na github.com
2. Nowe repo → wgraj plik index.html
3. Settings → Pages → Source: main → /root
4. Twój link: twójlogin.github.io/nazwarepo

Opcja 2 – Cloudflare Pages (błyskawiczny):
1. Zarejestruj się na pages.cloudflare.com
2. „Direct Upload" → przeciągnij plik HTML
3. Gotowy link HTTPS w 30 sekund

Opcja 3 – Vercel (kliknij i gotowe):
1. vercel.com → „Add New" → „Project"
2. „Browse" → wybierz folder z plikiem HTML
3. Deploy → gotowy link

Po otwarciu linku w Chrome Android:
Menu ⋮ → „Dodaj do ekranu głównego"
