JAK UZYSKAĆ GEMINI API KEY (bezpłatnie):

1. Wejdź na aistudio.google.com i zaloguj się kontem Google
2. Kliknij "Get API key" → "Create API key"
3. Skopiuj klucz zaczynający się od AIza...
